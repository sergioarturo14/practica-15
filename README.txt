DE:Grid Height, Grid Width, Pick a color
DS:Desing Canvas

1. Inico
2.  Mostrar título "Pixel Art Maker"
3.  Mostrar subtítulo "Choose Grid Size"
4. Mostrar formulario para seleccionar tamaño de la cuadrícula
5. Mostrar etiqueta "Grid Height" con campo de entrada de número (inputHeight) con valor mínimo de 1
6. Mostrar etiqueta "Grid Width" con campo de entrada de número (inputWidth) con valor mínimo de 1
7. Mostrar botón de enviar formulario
8. Mostrar enlace "imprimir!" para imprimir la página actual
9. Mostrar subtítulo "Pick A Color"
10. Mostrar selector de color (colorPicker)
11. Mostrar subtítulo "Design Canvas"
12. Mostrar tabla vacía (pixelCanvas) para diseñar el lienzo
13. Fin
